#### Описание:
<div align=center><img src="icon\icon_main.ico" width=30%></div>
<p>В рамках самостоятельной работы разработан pet-проект АИС учета коммунальных платежей. Главной задача проекта является централизованное хранение и сортировка всех квитанций клиентов, которые приходят в бумажном виде, а также проведение аналитики по различным типам расходов и количественным показателям. Эти задачи были необходимы для оптимизации процесса хранения и учета квитанций у пользователей.</p>
<p>Разработанный проект представляет собой систему учета квитанций коммунальных платежей, которая позволяет хранить, просматривать, осуществлять поиск по ключевым параметрам, проставлять метки (оплачено/не оплачено), редактировать и обрабатывать квитанции. Приложение имеет графическую оболочку, написанную на Pyside 2, а хранение данных производится в СУБД SQLite. Также в проекте встроена система аналитики данных, которая производит расчеты из базы данных и выводит аналитику по ключевым показателям в виде графиков и текстовых форм.</p>

<a id="a3"></a>
#### Результаты:
<p>По окончанию разработки, система была успешно протестирована (ручное тестирование, нагрузочное тестирование, функциональное тестирование). В ходе тестирования устранены следующие недочеты:
</p>

- На больших диагоналях экрана текст был нечитаемый (маленького размера). В результате введен функционал, который в ходе увеличения размера окна увеличивает текст;
- Устранены ошибки в системе аналитики. Несколько статистических данных приходили некорректными из-за неправильного алгоритма расчета ключевых параметров;
- Оптимизирован процесс поиска и сортировки в таблице (по ключевым словам).
